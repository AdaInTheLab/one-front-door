<section id="welcome" aria-label="Welcome to the docs">

Welcome to the One Front Door documentation. You're in the right place whether you're here to understand the philosophy, build your first page, or study the internals.

These docs are built with OFD itself. Every page you're reading passed the same habitability audit your pages will pass. The docs are the first test of the framework — if they're not habitable, nothing is.

</section>

<section id="start-here" aria-labelledby="start-here-heading">

## Start Here {#start-here-heading}

**If you want to understand why this exists:** Start with [Why This Exists](/docs/philosophy/why) — the origin story. Then read [One Front Door](/docs/philosophy/one-front-door) for the core principle, and [Habitability](/docs/philosophy/habitability) for what it means in practice.

**If you want to build something:** Jump to [Getting Started](/docs/guides/getting-started). You'll have a habitable page in five minutes.

**If you're not human:** Read [For Agents](/docs/philosophy/for-agents). It was written for you, not about you.

</section>
